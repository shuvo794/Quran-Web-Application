# আল-কুরআন ওয়েব অ্যাপ্লিকেশন - টেকনিক্যাল ডকুমেন্টেশন

এই ডকুমেন্টেশনে প্রজেক্টের আর্কিটেকচার, ডেটা ফ্লো এবং কিভাবে এটি কাজ করে তা বিস্তারিত আলোচনা করা হয়েছে।

## ১. প্রজেক্টের উদ্দেশ্য
এই অ্যাপ্লিকেশনটি একটি আধুনিক এবং দ্রুতগতিসম্পন্ন প্ল্যাটফর্ম যা ব্যবহারকারীদের কুরআন পড়তে, সার্চ করতে এবং নিজের পছন্দমতো ফন্ট ও সাইজ পরিবর্তন করতে সাহায্য করে।

## ২. ব্যবহৃত প্রযুক্তি (Tech Stack)
- **Frontend**: Next.js 16 (App Router) - এটি সার্ভার কম্পোনেন্ট এবং দ্রুত রেন্ডারিং নিশ্চিত করে।
- **Backend**: Hono (Node.js) - এটি একটি অতি দ্রুত ফ্রেমওয়ার্ক যা এপিআই হ্যান্ডেল করে।
- **Styling**: Tailwind CSS 4 - আধুনিক ডিজাইনের জন্য ব্যবহৃত হয়েছে।
- **Database**: লোকাল JSON ফাইল - এটি ডাটাবেস হিসেবে কাজ করে যা বিল্ডের সময় স্টাটিক পেজ জেনারেট করতে সহায়তা করে।

## ৩. ডেটা কিভাবে কাজ করে? (Data Flow)
প্রজেক্টটি একটি **Static Site Generation (SSG)** মডেলে তৈরি।
1. **ডেটা ফেচিং**: `scripts/fetch-data.js` স্ক্রিপ্টের মাধ্যমে `alquran.cloud` এপিআই থেকে ডেটা ডাউনলোড করে `data/` ফোল্ডারে সেভ করা হয়।
2. **সার্ভার সাইড জেনারেটর**: বিল্ডের সময় `app/surah/[id]/page.tsx` ফাইলটি ১১৪টি সূরার জন্য আলাদা আলাদা স্টাটিক HTML পেজ তৈরি করে।
3. **এপিআই লেয়ার**: Hono এপিআই (`app/api/[[...route]]/route.ts`) সার্চ এবং অন্যান্য ডাইনামিক রিকোয়েস্ট হ্যান্ডেল করে।

## ৪. প্রধান ফিচারসমূহ
- **থিম টগল**: ইউজার চাইলে ডার্ক এবং লাইট মোডে সুইচ করতে পারেন।
- **সেটিংস সেভ**: ফন্ট ফ্যামিলি এবং সাইজ পরিবর্তন করলে তা ব্রাউজারের `localStorage`-এ সেভ থাকে, যা পরবর্তী ভিজিটেও বজায় থাকে।
- **সার্চ ইঞ্জিন**: এটি সরাসরি লোকাল JSON ফাইলগুলো থেকে ডেটা সার্চ করে রেজাল্ট দেখায়, যা অনেক দ্রুত।

## ৫. মোবাইল রেসপন্সিভনেস
পুরো অ্যাপ্লিকেশানটি Tailwind CSS-এর গ্রিড এবং ফ্লেক্স সিস্টেম ব্যবহার করে বিভিন্ন স্ক্রিন সাইজের জন্য রেসপন্সিভ করা হয়েছে। সাইডবারটি মোবাইলে হাইড হয়ে যায় এবং ড্রয়ার হিসেবে ওপেন হয়।

## ৬. কিভাবে রান করবেন?
১. প্যাকেজ ইনস্টল করুন: `npm install`
২. ডেটা ডাউনলোড করুন: `node scripts/fetch-data.js`
৩. লোকাল হোস্ট রান করুন: `npm run dev`
